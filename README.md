# ระบบจัดทำเอกสารจัดซื้อ/จัดจ้างโรงเรียน

ระบบช่วยจัดทำเอกสารจัดซื้อ/จัดจ้างของโรงเรียน ตามระเบียบกระทรวงการคลัง
ว่าด้วยการจัดซื้อจัดจ้างและการบริหารพัสดุภาครัฐ พ.ศ. 2560
กรอกข้อมูลครั้งเดียว → ออกชุดเอกสารอัตโนมัติ พร้อมรันเลขทะเบียนและทะเบียนคุม

ทำงานแบบ **offline** บนเครื่องเดียว ข้อมูลทั้งหมดเก็บในไฟล์ `data/school.db`
(สำรองข้อมูลง่าย ๆ แค่ก๊อปปี้ไฟล์นี้)

---

## สถานะการพัฒนา

- ✅ **เฟส 1 (MVP)** — ทำเสร็จแล้ว
  - ตั้งค่าข้อมูลโรงเรียน
  - จัดการผู้ขาย/ผู้รับจ้าง
  - สร้างเรื่องจัดซื้อ + รายการพัสดุ + รันเลขทะเบียนอัตโนมัติ (รีเซ็ตทุกปีงบ)
  - ออกเอกสาร Word: **รายงานขอซื้อขอจ้าง**, **ใบตรวจรับพัสดุ**
  - ทะเบียนจัดซื้อจัดจ้าง (ดาวน์โหลด Excel)
  - แปลงจำนวนเงินเป็นข้อความไทย (บาทถ้วน) อัตโนมัติ
- 🚧 **เฟส 2 (กำลังทำ)** — แปลงฟอร์มจริงเป็นแม่แบบ docxtpl + ข้อมูลครบขึ้น
  - ✅ data model ใหม่ (บุคลากร/ฝ่าย/โครงการ/คณะกรรมการ + ฟิลด์ราคากลาง/ส่งมอบ/ค่าปรับ)
  - ✅ ระบบรันเลข 4 ชุดแยก (บันทึก/คำสั่ง/ใบสั่งซื้อ/ใบสั่งจ้าง) แบบเสนอเลข+แก้ได้+bump
  - ✅ ผู้ตรวจรับคนเดียว (โดยอนุโลม) vs คณะกรรมการ — เลือก/แนะนำตามวงเงิน
  - ✅ มาสเตอร์ลิสต์ + dropdown จำค่าเดิม
  - ✅ แม่แบบ: รายงานขอซื้อ/จ้าง (+แนบท้าย), ใบตรวจรับพัสดุ (+เสนอผล+เบิกจ่าย) — ตราครุฑ, หัวกระดาษมาตรฐาน, รองรับคนเดียว/คณะกรรมการ
  - ⏳ แม่แบบที่เหลือ: คำสั่งแต่งตั้ง, รายงานผลพิจารณา, ใบสั่งซื้อ/จ้าง, ใบเสนอราคา, ประกาศผู้ชนะ, TOR, ใบส่งมอบงาน
  - ⏳ งบประมาณ/เบิกจ่าย (ตัดยอด)
- ⏳ เฟส 3 — ทะเบียนครุภัณฑ์ + ค่าเสื่อม + Dashboard ติดตามสถานะ
- ⏳ เฟส 4 — แพ็กเป็น .exe สำหรับขาย + license
- ⏳ เฟส 5 — วิธีคัดเลือก / e-bidding / งานก่อสร้าง (ปร.4/5/6)

## วิธีเริ่มใช้งาน (สำหรับเครื่องที่ติดตั้งแล้ว)

ดับเบิลคลิกไฟล์ **`เริ่มโปรแกรม.bat`** — เบราว์เซอร์จะเปิดที่หน้าโปรแกรมเอง
(อย่าปิดหน้าต่างสีดำขณะใช้งาน เมื่อเลิกใช้ให้ปิดหน้าต่างนั้น)

---

## วิธีติดตั้งบนเครื่องใหม่ (สำหรับนักพัฒนา)

1. ติดตั้ง **Python 3.12** (https://www.python.org/downloads/ — ติ๊ก "Add to PATH")
2. เปิด PowerShell ที่โฟลเดอร์นี้ แล้วรัน:
   ```powershell
   python -m venv .venv
   .venv\Scripts\python.exe -m pip install -r requirements.txt
   ```
3. รันโปรแกรม:
   ```powershell
   .venv\Scripts\python.exe -m app.main
   ```
   หรือสำหรับพัฒนา (รีโหลดอัตโนมัติเมื่อแก้โค้ด):
   ```powershell
   .venv\Scripts\python.exe -m uvicorn app.main:app --reload
   ```
4. เปิดเบราว์เซอร์ที่ http://127.0.0.1:8000

> **ฟอนต์เอกสาร:** เอกสาร Word ใช้ฟอนต์ `TH Sarabun New` (ฟอนต์มาตรฐานราชการ)
> หากเครื่องยังไม่มี ให้ติดตั้งฟอนต์นี้ก่อนเพื่อให้เอกสารแสดงผลถูกต้อง

## โครงสร้างโปรเจกต์

```
app/
  main.py              จุดเริ่มโปรแกรม + เปิดบราวเซอร์
  database.py          เชื่อมฐานข้อมูล SQLite
  models.py            ตารางข้อมูล (โรงเรียน, ผู้ขาย, เรื่องจัดซื้อ, ฯลฯ)
  thai_utils.py        ฟังก์ชันไทย (บาทถ้วน, วันที่ไทย, ปีงบประมาณ)
  services/
    doc_number.py      ตรรกะรันเลขทะเบียนอัตโนมัติ
    docgen.py          สร้างเอกสาร Word
    register_export.py ส่งออกทะเบียนเป็น Excel
  routers/pages.py     เส้นทางหน้าเว็บทั้งหมด
  templates/           หน้าเว็บ (HTML)
data/
  school.db            ฐานข้อมูล (1 ไฟล์ต่อโรงเรียน — สำรองด้วยการก๊อปปี้)
  documents/           ไฟล์เอกสารที่สร้าง
```
