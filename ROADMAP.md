# D-Doc Roadmap

> ระบบจัดการเอกสารและพัสดุโรงเรียน
> กลยุทธ์: **"เรียบง่าย เร็ว เฉพาะงานเอกสารจัดซื้อ"** สำหรับคนที่รู้สึกว่าคู่แข่ง (EGPEASY) ซับซ้อนเกินไป
> + ออฟไลน์/ซื้อขาด/ข้อมูลเป็นของโรงเรียนเอง

## หลักการที่ห้ามทิ้ง (จุดแข็งที่ชนะคู่แข่ง)
1. **ไม่ทำฟอร์มยักษ์** เก็บการแยกขั้นตอน: สร้างเรื่อง → เติมเลข/วันที่ทีหลัง
2. **ฟิลด์ไม่จำเป็น = optional + มีแบนเนอร์บอก** ว่าอันไหนไม่ต้องกรอก
3. **เริ่มใช้ได้เร็ว** ไม่บังคับตั้งค่ามาสเตอร์หลายชุดก่อน (memo ออกเลขอัตโนมัติ, มาสเตอร์เลือกได้ทีหลัง)
4. **ออกทั้งชุดเป็นไฟล์ Word เดียว** เรียงตามขั้นตอน

## เฟส 2.5 — ปิดช่องว่างด่วน (จาก EGPEASY) โดยคงความง่าย ✅ เสร็จแล้ว
- [x] **VAT (มี/ไม่มีภาษีมูลค่าเพิ่ม)** — toggle ต่อเรื่อง + แยกราคาก่อนภาษี/VAT ในใบเสนอราคา
- [x] **เลือกผู้ลงนามในใบสั่งซื้อ/จ้าง** (ผอ. / หัวหน้าเจ้าหน้าที่)
- [x] **ค่าปรับอัตโนมัติ** — "ส่งเกินกำหนด X วัน" → ค่าปรับ = อัตรา% × วงเงิน × วัน แสดงในใบตรวจรับ
- [x] **คัดลอกเรื่อง (Duplicate)** — clone เรื่องเดิมเป็นร่างใหม่ (เลขบันทึกใหม่ ไม่ก๊อปเลข/วันที่)
- [x] **"ใช้วันที่เดียวกันทั้งหมด"** — ปุ่มเติมวันที่ทุกช่องพร้อมกัน
- [x] **ใบส่งของ เล่มที่/เลขที่** — ฟิลด์ในกลุ่มตรวจรับ

## เฟส 2.6 — ครบเครื่องงานจัดซื้อ
- [ ] **ประเภทพิเศษ: ว.804 (≤50,000), ว.119, จำเป็นเร่งด่วน** — ปรับถ้อยคำ/ข้ออ้างกฎหมายในเอกสารตามประเภท
- [ ] **Dashboard กราฟ** — กราฟจัดซื้อรายเดือน + แยกตามประเภท (ใช้ไลบรารีกราฟแบบ offline หรือ SVG เอง)
- [ ] **ส่งออก PDF** ปุ่มเดียว (นอกจาก .docx) สำหรับพิมพ์
- [ ] **สรุปผลการจัดซื้อจัดจ้างรายเดือน** (รายงาน/Excel)
- [ ] เลือกวิธีคัดเลือก / e-bidding (ปรับชุดเอกสาร)

## เฟส 3 — ขยายเป็น "บริหารพัสดุ" (ระบบที่เหลือ — ทำทีหลัง)
- [ ] **ทะเบียนครุภัณฑ์ + ค่าเสื่อมราคาอัตโนมัติ** + เลขครุภัณฑ์ + ใบยืมพัสดุ
- [ ] **บัญชีวัสดุ** (รับเข้า-เบิกจ่าย) + **ใบเบิกพัสดุ** (สร้างอัตโนมัติจากเรื่องจัดซื้อ)
- [ ] **ตรวจรับวัสดุ → ลงบัญชีวัสดุอัตโนมัติ**

## เฟส 3.5 — โมดูลเฉพาะทาง
- [ ] **ตรวจสอบพัสดุประจำปี** + สอบสวนข้อเท็จจริง + จำหน่ายครุภัณฑ์
- [ ] **อาหารกลางวันนักเรียน** (จ้างเหมา / ซื้อวัตถุดิบ / ซื้อ+จ้างแม่ครัว)
- [ ] **บัญชีหนังสือเรียน** (นำเข้า Excel + ใบเบิก + ทะเบียน)

## เฟส 4 — ทำเป็นสินค้า
- [ ] แพ็กเป็น `.exe` (PyInstaller) + ตัวติดตั้ง — รวม static/, doc_templates/, fonts
- [ ] ระบบ license / activation ต่อโรงเรียน
- [ ] สำรอง/กู้คืนข้อมูล (ก๊อปไฟล์ .db)
- [ ] ตั้งราคา/แพ็กเกจ (จุดขาย: ซื้อขาด + ออฟไลน์ + ข้อมูลเป็นของโรงเรียน)

## สิ่งที่คู่แข่ง (EGPEASY) มี แต่เราจงใจ "ไม่ทำ" เพื่อคงความง่าย
- ไม่ยัดทุกอย่างเป็น modal เดียว 40+ ช่อง
- ไม่บังคับตั้งมาสเตอร์ 9 ชุดก่อนเริ่ม
- ไม่ทำเมนูลึกหลายชั้นจนต้องมีคู่มือ "ตัวอย่างการกรอก"
